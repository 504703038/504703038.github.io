1、安装

  • Windows
  • Ubuntu/Debin

下载

官网下载安装包，可以只下载 Server 或者下载 Neo4j Desktop 版本（Server 类似精简版，Desktop 有UI界面以及其他工具）。

下载 Community Server 版本，将下载后的 zip 文件解压并给目录下的 bin 文件夹添加环境变量（解压路径/neo4j-community-版本号/bin

安装

==执行Neo4j命令不建议使用PowerShell==

添加完路径后在控制台执行 neo4j install-service 命令安装 neo4j 服务。安装完成后执行 neo4j status 命令可以查看 neo4j 运行状态。

安装Neo4j

配置Neo4j

将 dbms.connectors.default_listen_address=0.0.0.0 取消注释即可进行远程访问

远程访问Neo4j管理页面，使用浏览器访问 http://设备IP:7474

2、管理页面

浏览器访问 http://localhost:7474 ，默认账号密码为 neo4j 。

3、连接数据库

Neo4j 的默认连接 url 为 neo4j://localhost:7687

2、查

按标签名称查询并返回查询到的节点的属性值

语法元素 描述
MATCH Neo4j CQL 命令
<node-name> 命名查询得到的节点
<label-name> 要查询节点的标签名称
RETURN Neo4j CQL 命令
<Property1-name>…<Propertyn-name> 节点的属性的名称
