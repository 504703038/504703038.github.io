树莓派

发表于 2020-10-08 分类于 Raspberry Pi
本文字数： 1.1k 阅读时长 ≈ 1 分钟

1、相关设备信息

Raspberry Pi : 4B(2G)
TF卡 : 32G
电源 : 5.1V/3A

2、系统安装

1、系统映像下载

在 官网下载地址 下载 Raspberry Pi OS

选择含推荐软件的桌面版

下载完成后解压 zip 文件得到 后缀名为 img 的系统映像文件

2、将映像写入TF卡

首先将TF卡格式化

使用 Win32DiskImager 工具将映像写入TF卡(Win32DiskImager下载)

完成后TF卡会有两个分区

系统默认关闭了 SSH 连接，想要使用 SSH 需要在 boot 分区中新建空白文件 ssh (注意：没有后缀名)

3、运行

1、无屏幕

个人比较穷所以没有买屏幕

0、工具
  • 一台PC(废话)
  • Advanced IP Scanner
  • PuTTY
  • VNC Viewer
1、准备工作

将之前准备好的TF卡插入树莓派并接通电源，给树莓派连接网络(让PC与树莓派处于同一局域网内)。

2、获取树莓派IP

使用 Advanced IP Scanner 扫描IP地址，找到树莓派的地址

当成功使用PuTTY连接到树莓派后就可以给树莓派分配静态IP地址，之后就不需要再扫描IP地址了

配置完后重启树莓派

3、使用PuTTY连接树莓派

使用 PuTTY 连接树莓派，如果不确定那个是树莓派的IP地址就挨个尝试一下。。。。。

如果连接成功会提示你输入登录用户名以及密码

登录账号及密码

连接成功界面

4、使用远程桌面访问树莓派

先使用 PuTTY 连接树莓派安装 xrdp

可能因为网络原因需要切换源，后面再补吧

运行 xrdp

在 PC 上启动 远程桌面连接

输入密码

成功连接到树莓派

5、使用VNC访问树莓派

1、开启树莓派VNC功能

2、连接树莓派

输入树莓派IP地址进行连接，输入账号密码登录

3、遇到的问题

VNC Viewer 无法显示界面

解决方法：修改树莓派的分辨率

重启树莓派
